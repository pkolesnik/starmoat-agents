# Capital Project Accounting  `fin.capital_project_accounting`

> function: `finance_ops` · value stage: Business Operations · optional · stage: scale · default execution: `hybrid`

**Why you need this:** Tracks spend and returns on capital projects so investments are capitalized and measured correctly.

**What good looks like:** Every capital project has a budget, tracked spend, and a measured post-completion return.

## KPIs
- budget adherence %
- projects with measured ROI %
