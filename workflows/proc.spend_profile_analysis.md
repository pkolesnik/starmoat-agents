# Analyze Spend Profile & Sourcing Opportunities  `proc.spend_profile_analysis`

> function: `procurement` · value stage: Business Operations · optional · stage: series_a · default execution: `hybrid`

**Why you need this:** Build visibility into where money is spent to find consolidation and savings opportunities.

**What good looks like:** Spend is classified by category and supplier with quarterly savings opportunities surfaced and tracked.

## KPIs
- spend classified %
- identified savings $
