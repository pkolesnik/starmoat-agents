# Manage Supplier Contract Lifecycle  `proc.contract_lifecycle`

> function: `procurement` · value stage: Business Operations · optional · stage: series_a · default execution: `hybrid`

**Why you need this:** Track contract terms, renewals, and obligations across the supplier base.

**What good looks like:** All active contracts are in a repository with renewal alerts and no surprise auto-renewals.

## KPIs
- contracts with renewal alerts %
- missed renewal count

## Powered by (candidate agents)
- `contract-renewal-tracker`
