# Budget Variance Analysis  `fin.budget_variance_analysis`

> function: `finance_ops` · value stage: Business Operations · optional · stage: series_a · default execution: `hybrid`

**Why you need this:** Explains actuals versus plan so leaders can correct course on spend and revenue.

**What good looks like:** Every material variance has a named owner and a documented root cause each month.
